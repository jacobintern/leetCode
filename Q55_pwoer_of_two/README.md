# Power of Two(Easy)

## Beginners Guide

Given an integer n, return true if it is a power of two. Otherwise, return false.

An integer n is a power of two, if there exists an integer x such that n == 2x.

Example 1:
---
```go=
Input: n = 1
Output: true
Explanation: 20 = 1
```

Example 2:
---
```go=
Input: n = 16
Output: true
Explanation: 24 = 16
```

Example 3:
---
```go=
Input: n = 3
Output: false
```

Rules:
---
* $-2^31 <= n <= 2^31 - 1$
